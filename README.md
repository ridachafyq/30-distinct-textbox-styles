# CSS Textbox Collection

A single-page project that showcases **30 different textbox (input) styles** created using **pure HTML and CSS**.

Each textbox demonstrates different UI techniques such as focus effects, borders, shadows, gradients, glass effects, and underline styles.
This project serves as a **design reference and learning resource for modern CSS input styling**.


## ✨ Features

* 🎨 30 unique textbox/input designs
* ⚡ Pure CSS (no JavaScript)
* 📱 Responsive two-column layout
* ♿ Accessible focus states using `:focus-visible`
* 🌙 Dark mode support with `prefers-color-scheme`
* 🧩 Easy to reuse in forms and UI components
